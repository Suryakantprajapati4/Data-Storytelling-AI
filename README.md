# 📊 Data Storytelling AI

Transform raw business data into meaningful insights, visualizations, reports, and AI-generated narratives.

## 🚀 Overview

Data Storytelling AI is an interactive analytics application built with Python and Streamlit that helps users:

- Upload CSV/Excel datasets
- Analyze business KPIs automatically
- Generate interactive visualizations
- Get AI-powered insights
- Create business story reports
- Export reports as PDF
- Chat with data using natural language

The goal of this project is to bridge the gap between raw data and business decision-making by converting data into easy-to-understand stories.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/DataStorytellingAI.git

cd DataStorytellingAI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
